# Community Survey Topic — Agentic Workflow Guidelines

**Created:** 2026-04-19

## Why tracked
To collect and keep current the canonical guidelines and design principles published by frontier AI labs (Anthropic, OpenAI, Google/Gemini) on how to build agentic workflows. Scope is intentionally constrained to first-party guidance from these three labs — no community/aggregator sources — so the resulting reference reflects the recommendations of the model providers themselves.

## Audience
Personal reference for designing and building agentic systems.

## Lens
Extract canonical, actionable design principles and patterns I can apply when building agents — emphasizing what each lab explicitly recommends (and warns against) for tool use, planning, evaluation, and safety in agentic workflows.

## Scope constraints (project-specific)
- **Sources:** ONLY official Anthropic, OpenAI, Google/Gemini properties (engineering blogs, research blogs, docs sites, cookbooks, model cards/system prompts when published officially).
- **Excluded:** Hacker News, ProductHunt, Medium/Substack, X/Twitter, Quora, third-party blogs.
- **Why:** The user wants the *labs' own design guidance*, not community discourse.
