# Learning the Latent Causal Structure for Modeling Label Noise

**Authors:** (NeurIPS 2024 proceedings)
**Code:** Available via paper supplementary

## Problem

The noise transition matrix T(x) = P(ỹ|y,x) is instance-dependent in practice, but existing methods either (1) estimate T as a global constant (ignoring instance-dependence) or (2) estimate T(x) via neural networks without principled structure (no identifiability guarantees). Neither approach captures the **latent causal mechanism** that generates label noise — the confounders and mediators that determine why a specific instance gets a specific noisy label.

## Method: CSGN (Causal Structure for Generating Noise)

**Causal generative model:**

The key insight: label noise is generated by a causal process involving latent variables Z with a DAG (directed acyclic graph) structure. The noisy label ỹ depends not directly on x, but on latent causal variables that mediate the relationship.

**VAE architecture with learnable sparse causal masks:**

```
Encoder: x → Z (with sparse mask MX controlling which X-features affect Z)
Decoder 1: Z → y (clean label prediction)  
Decoder 2: Z, y → ỹ (noisy label generation via transition matrix)
Sparse masks: MX (input→latent), MỸ (latent→noisy label)
```

**Learnable sparse masks:** MX and MỸ are binary masks learned via continuous relaxation (Gumbel-sigmoid). The sparsity of MỸ encodes which latent causal variables causally affect the noise process — directly identifies the causal structure.

**Joint ELBO training:**

```
L = E[log p(ỹ|Z,y)] + E[log p(y|Z)] - KL[q(Z|x) || p(Z)] + λ·(||MX||_0 + ||MỸ||_0)
```

The sparsity penalty enforces discovery of the minimal causal skeleton — prevents overfitting to spurious correlations in noise patterns.

**T(x) derivation:** The noisy label decoder p(ỹ|Z,y) implicitly defines T(x) = P(ỹ|y, Z=f(x)), marginalized over the posterior Z.

## Key Results

| Dataset | CSGN | Best Baseline | Delta |
|---------|------|--------------|-------|
| CIFAR-10 IDN-50% | **95.88%** | DivideMix 86.30% | +9.58pp |
| CIFAR-100N (real) | **71.99%** | CoDis ~70% | +~2pp |
| WebVision top-1 | **79.84%** | CrossSplit 79.19% | +0.65pp |
| CIFAR-10N Worst | ~91% | SOP+ 94.0% | competitive |

The CIFAR-10 IDN result (+9.58pp over DivideMix) is exceptional — confirms that structured, instance-dependent noise is where causal modeling provides the largest advantage.

## Relevance to Proxy Label Learning

**Rating: Very High (best-fit framework for attribution-derived proxy labels).**

1. **Attribution noise IS causally structured:** Shapley values depend causally on feature interactions, model internals, and background distribution choice. The causal graph structure (which features cause the noise, via which latent mechanism) is exactly what CSGN's sparse masks MX, MỸ are designed to discover.
2. **VAE latent space = attribution uncertainty:** The latent Z in CSGN can be interpreted as a compressed representation of the factors that determine attribution instability. Learning this latent structure is the learning-theoretic analog of identifying when Shapley values are reliable.
3. **DAG structure = feature interaction graph:** For tabular proxy labels (e.g., user-feature interactions in recommendation), the learned DAG over latent variables provides an interpretable map of which feature dimensions causally influence label corruption.
4. **Strongest on IDN — matches proxy label structure:** Proxy labels from attribution models are instance-dependent by construction. CSGN's +9.58pp on IDN-50% (vs DivideMix) directly demonstrates the advantage of causal modeling in this regime.
5. **Limitation:** VAE + sparse mask learning adds training complexity (hyperparameters: mask sparsity penalty λ, Gumbel temperature). End-to-end training may be unstable for high-dimensional feature spaces typical in production recommendation models.
